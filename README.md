## End to End ML project
# Diamond Price Prediction, Clarity Classification & Customer Segmentation

## Project Overview
This project uses **machine learning models** to predict **diamond price** (regression) and **clarity** (classification) based on physical and quality features of diamonds.  
Additionally, it performs **customer segmentation** to understand patterns in the diamond market.  
The goal is to help **jewelers, retailers, and buyers** make **data-driven decisions** for buying, selling, and pricing diamonds.

## Features Used
- **Numerical:** carat, depth, table, x, y, z  
- **Categorical:** cut, color, clarity (for classification)  
- **Targets:** Price (regression), Clarity (classification)

## Prerequisites
- Python 3.11.0+  
- Libraries:  
  - pandas  
  - numpy  
  - scikit-learn  
  - matplotlib / seaborn  
  - Flask (for the web app)  
  - joblib (for saving/loading models)
